# Petshop Patinhas Felizes 

O projeto consiste em um CRUD (Create, Read, Update, Delete) escrito em PHP puro, focado em aplicar os conceitos de **Programação Orientada a Objetos (POO)** e o padrão de projeto **Singleton** para a conexão com o banco de dados.

## Como executar o projeto

Para rodar este projeto localmente, você precisará ter o **XAMPP** instalado.

1. **Clone o repositório** diretamente dentro da pasta `htdocs` do seu XAMPP

2. **Inicie o servidor:** Abra o Painel de Controle do XAMPP e clique em "Start" nos módulos **Apache** e **MySQL**.

3. **Acesse o Sistema!**
   Com o banco configurado, vá para a tela inicial do sistema acessando:
   👉 `http://localhost/crud-pet-shop/index.php`

   Se você colocou em uma nova pasta no htdocs, digite:
   👉 `http://localhost/nome_da_sua_pasta/crud-pet-shop/index.php`
